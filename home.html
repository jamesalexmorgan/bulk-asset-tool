With the BAT we can...

Gather

  • Load up any number of assets into the table and view all the available attributes and metadata
  • Track down and organise assets using the powerful searching and multiple sorting tools
  • Build custom CSVs with specific fields

Create

  • Bulk create assets of any type using a number or a CSV of names
  • Create a whole site structure from an IA CSV specifying different asset types!

Modify

  • Easily fire off any js api function on any number of assets
  • Execute a bulk CSV change, modifying multiple attributes and metadata values from any number of assets with one click!
